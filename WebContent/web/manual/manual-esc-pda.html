Record an Inspection with a PDA

To record and complete an inspection with a PDA or handheld device:

  1. Select Internet Explorer from the Start menu on your PDA.
  1. In the address line at the top of the screen, type the web address for your PermiTrackESC site (https://MyNPDESPermit.com/).
  2. Tap the Go icon.
  3. On the Welcome page, tap the PermiTrackESC link near the top of the page.
  4. Tap the Login link near the top of the page.
  5. On the Login screen, type your user name and password.
  6. Tap the Login button.
  7. If the system displays a message warning you about sending information over the Internet, tap the Yes button.
  8. On the next page, click the PermiTrackESC link.
  9. On the list of projects, tap the Inspection icon next to the name of the project you are inspecting.
  10. On the inspection page, record information about the inspection.
    (See the PDA Inspection page on page for more information.)
  11. If you want to attach a photo:

    a) Take the photo with your PDA.

    b) Scroll to the appropriate BMP or Inspection Item.

    c) Tap the Add Photo link.

    d) Tap the Browse button to locate the photo on your PDA.

    e) Tap the name of the photo you want to upload.
  12. Select Draft from the Save As drop-down list at the top or bottom of the page if you have not completed the form.

    Note : You want to be sure the inspection is final before you save it as Final. When the inspection is complete, you can select Final in this drop-down list before saving.
  13. Click the Save As button to save the inspection.
  14. Click the OK button on the success message.

Later, you can view or edit this inspection record from a computer.

PDA or Wireless Handheld Device

For convenience and to capture timely data, an inspector can log inspections at the project site using a PDA or handheld wireless device. Through a web connection from the PDA, the inspector logs in directly to the PermiTrackESC website. This login lets the inspector enter the inspection data, upload pictures taken with the handheld device, record notes, and save the inspection as a draft or final version. Back at the office, the inspector can review the inspection information or edit a draft inspection.

Note: At this time, PermiTrackESC supports only the Palm® Treo TM 700w/wx Smartphone . For support of other devices, please contact your PermiTrack Administrator.

PDA Project List

The Project List page shows all of your construction projects, along with a link for starting an inspection of the project. From this page, you can view more detailed information for a project or record an inspection.

The Projects List opens when you log in to the PermiTrackESC program on the PDA.

Note: For more information on projects, see Projects & Inspections .

Columns and Options on the Project List page
  1. Sort By : Use the Sort By field to arrange the list of projects by the permit number, status, other characteristic. Select an item from the Sort By drop-down list. The Project List now includes a column on the left that lists the value in that category for each project.
  2. Project Name : Tap the underlined project name to view detailed information about that project.
  3. Options : Tap an icon in the Options column to inspect a site or view the site inspection history.

Inspect : Tap the Inspect icon next to a project name to record an inspection for that site. The inspection can include notes and photos. The PDA Inspection page () opens.

Inspection History : Tap the Inspection History icon next to a project name to see the inspection history for that site. The PDA Inspection History page () opens.

Reload : If a project is not listed, but you expect it to appear, your list may need to be refreshed. Click the Reload icon to refresh the list of projects.

PDA Inspection page

On the New Inspection page, you record an inspection of the project, record pertinent weather data, make notes about compliance with the BMPs, and, if needed, upload a photo to include in the inspection record.

The New Inspection page opens when you tap the Inspect icon next to a project name on the Project List page or when you tap Inspect link on the View Project Information page.

Fields on the New Inspection page
Project Information: Permittee, Project Documents, Contact Information

The upper area of the page displays the project information so that you can confirm that you are working with the correct project.

Project Inspection BMPs

Below the project information is a list of BMPs:

  1. BMPs : Each BMP required for this project is listed in the Project Inspection BMPs area. In this area, review the status of each control measure to complete your inspection. There is space to record your findings further down this page.

Inspection Information

When recording a new inspection, enter the following information:

  1. Inspection Date: Enter the date when you performed the inspection. Use mm/dd/yyyy format:

    mm = two-numbers to indicate the month, for example 09.
    dd = two-numbers to indicate the day of the month, for example 01.
    yyyy = Four numbers for the year, for example 2007.
  2. Weather Trends : Enter information about recent weather that may contribute to a problem with the erosion control or may require additional monitoring or action.
  3. Last Precip. End Date : Type the date for the last day of the most recent precipitation, if pertinent.
  4. Last Precip. Amount : Enter the amount of rainfall, in inches, from the most recent precipitation.
  5. Source of Data : Select the source of the precipitation data entered above.
  6. Temperature : Record the temperature in F (Fahrenheit) at the time the inspection was made.
  7. Reason for Inspection : Select the reason this inspection is being performed.
Inspection Items

The Inspection Items area of the page lists the BMPs to be inspected:

  1. Inspected : Tap the circle next to Yes or No to indicate whether this BMP has been inspected.
  2. Action Required : Select a follow-up action, if one is required. For example, the inspection may result in a phone call if the inspector found a problem.

Note : This field is available only to the permit authority.

  1. Comment : Enter any comments you want to add to the inspection report.
Inspection Items

The Inspection Items area of the page lists all of the BMPs associated with the project. If a BMP is required, you must record your observations and update the status.

  1. BMP : Name of the BMP or control measure to be inspected.
  2. Description: Description of the BMP to be inspected. This description is entered during project setup to provide specific information for the inspector (e.g., details specific to that item on that project).
  3. Required : Indicates whether the BMP is required for this project.
  4. Inspected : Indicate whether this item was inspected during the visit to the project site.

Yes : Select Yes if this control measure was checked or inspected during this inspection.

No : Select No if this control measure was not checked or inspected during this inspection.

  1. Status : Status of the BMP at the time of the inspection. The status selected here is automatically entered on the inspection form for the next inspection.

Active : The BMP is active and should be implemented at this project site.

Inactive : The BMP does not apply to the project site at this time or at this phase of construction.

Completed : Use of this BMP is complete. For example, the project has progressed beyond the need for this BMP, so it is no longer needed at this project site.

  1. Condition : Select the condition from the drop-down list to indicate the effectiveness of the BMP application.

N/A : Not Applicable. The BMP does not apply to the project site at this time.

Correct : This control measure is being used or applied correctly at the project site.

Not Applied : This control measure should in place at the site, but it is not being used at this time.

Ineffective : This control measure is in place at the project site, but it is not effective in providing erosion control.

  1. Comment : Enter your comments about the application of this BMP, its condition, or more detailed information.
  2. Add Photo : Take a photo with your PDA, and then tap the Add Photo link. Use the Browse button to locate the photo on your PDA.

Note : To add a photo to the inspection record, it must be included under a BMP.

  1. Add BMP : Tap the Add BMP link to add a new BMP.
Save As Field and Button
  1. Save As field: Select a Save As option from the drop-down list to save the inspection as a draft to be modified later or as the final record of the inspection.

Draft: The inspection record is not complete. To make changes, edit the inspection record through the website from a computer.

Draft inspections can be viewed by PermiTrackESC users who have access to your project website, but cannot be viewed through the interactive map interface.

Final: The inspection record is complete. No changes can be made after the inspection is final.

Final inspection records are immediately viewable by the general public through the public-access interactive maps.

  1. Save As button: Tap the Save As button to save your inspection record.

PDA Inspection History page

The Inspection History page lists all of the inspections recorded for the project you selected from the list of projects. Use this page to review the list of inspections or to open a draft inspection for editing.

The Inspection List page opens when you tap the Inspection History icon next to a project name on the Project List page ().

Icons and Links on the Inspection History page

Icons and Options :

  1. Inspect : Tap the Inspect icon to open the PDA Inspection page to record an inspection for this project.
  2. Edit : Tap the Edit icon next to an inspection date to edit the inspection on that date. The inspection can include notes and photos.

Note : You can only edit a draft inspection. If the inspection was saved as Final, it can be viewed, but not edited.

  1. Back : Return to the PDA Project List page.
  2. Inspection Date : Click on an inspection date to view all of the information recorded for that inspection. The Inspection View page opens ().

PDA Inspection View page

The Inspection View page opens when you tap the inspection date link on the Inspection History page ().
