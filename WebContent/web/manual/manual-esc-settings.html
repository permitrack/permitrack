Settings

Individual users have access to the Administration application in order to change their password and contact information.

When you log in to the PermiTrack Client Access Point, it opens to one of your application pages. For example, if you use PermiTrack-ESC, it opens to the PermiTrack-ESC opening page. The applications available to you are shown in the tabs near the top of the screen.

To administer your account, click the Administration tab.

Figure 44. Client Access Point Opening page

View User page

The View User page displays all of your contact and account information. From this page, you can update this information or change your login password.

The View User page opens when you click the Administration tab on the opening page ().

Figure 45. View User page

Links on the View User page

  1. Edit User : Click the Edit User link to edit your address, phone number, or e-mail address. When you click this link, the Edit User page opens (on page ).
  2. Change Password : Click the Change Password link to change your password for logging in to the Client Access Point. The Change Password page () opens.

Edit User page

The Edit User page lets you change your contact information, including address, phone, and e-mail address.

The Edit User page opens when you click the Edit User link on the View User page ().

Figure 46. Edit User page

Fields on the Edit User page

  1. User Information : Update your name or address in this section.
  1. User Name (view only): The user name you enter when you log in to the Client Access Point. (This name is set by your Client Administrator.)
  2. First Name & Last Name : Your first and last name, as it should appear in lists in the applications.
  3. Title : Your job title or position.
  4. Acct. Type (view only): You are assigned to a User account. This information is set by your Client Administrator.
  1. Contact Information : Update your address and/or phone information in this section.
  1. Address 1 & Address 2 : Lines 1 and 2 of your mailing or street address.
  2. City, State, & Zip : City, state, and zip code for your mailing or street address.
  3. Email : Your email address.
  4. Primary Phone : Your primary or preferred phone number.
  5. Secondary Phone : Your secondary phone number. For example: a cell phone.
  6. Fax Phone : Phone number where you receive faxes.

Buttons on the Edit User page

  1. Update User : Click the Update User button to save the changes you made on this page and return to the View User page ().
  2. Cancel : Click the Cancel button to discard any changes you made on this page and return to the View User page (.) Your changes are not saved.

Change Password page

The Change Password page lets you change your password for logging in to the Client Access Point. Before changing the password, confirm that the correct user name is displayed in the Change Password box on this page.

A password can be modified by either the user or the Client Administrator.

The Change Password page opens when you click the Change Password link on the View User page ().

Figure 47. Change Password page

Fields on the Change Password page

  1. New Password : In the New Password field, type the new password that you want to use. You do not enter your old or existing password.
  2. Confirm : In the Confirm field, re-type the new password to confirm that you entered it correctly.

Note :When you click the Change Password button below, the new password is set. You will use this new password the next time you log in.

Buttons on the Change Password page

  1. Change Password : Click the Change Password button to save the new password that you entered on this page. You return to the View User page (.)
  2. Cancel : Click the Cancel button to discard any changes you made on this page and return to the View User page (.) Your changes are not saved.
