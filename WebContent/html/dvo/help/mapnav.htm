Map Navigation Tools

 

  Zoom In

Use this tool to zoom in on the map. A single click on the map will zoom in centered on the click. You can also drag a box to define the area to zoom in to.

 

  Zoom Out

Use this tool to zoom out on the map. A single click on the map will zoom out centered on the click. You can also drag a box to define the area and scale to zoom out from (a smaller box will zoom out further)

 

  Zoom to Full Extent

This button will refresh the map to the full data extent.

 

  Zoom Previous

This button will return the map to the previous extent.

 

  Pan

Use this tool to shift the view (pan) on the map. A single click on the map will re-center the map on the click. You can also drag the map to the extent you wish.
