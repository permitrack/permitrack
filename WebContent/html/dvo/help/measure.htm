Measurement Tools

 Measure Tool

Use this tool to measure a segment of line. Single left click on the map to start measuring. A second left click on the map will end the segment. Click the "Clear Selection" button to clear the measurements.

Set Map/Measure Units

Use this feature to set the map and measure units. Click on the mapping options pull-down. Select 'Set Map/Measure Units'. To change the map units, click on the pull-down below 'Display Units'. Select the units of measure that you want and click on 'Set Units'.
